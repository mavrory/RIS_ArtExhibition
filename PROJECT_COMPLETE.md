## 🎯 Основные достижения

### 1. Реалистичная система оплаты
Как в Steam, App Store, Amazon:
- Пополнение кошелька из внешних источников
- Покупка только через кошелек
- Прозрачная история операций

### 2. Удобный UX
- Автообновление баланса
- Понятные сообщения об ошибках
- Кнопка "Add Funds" при недостатке средств
- Нет дублирования элементов

### 3. Полная прозрачность
Каждая транзакция имеет смысл:
- **DEPOSIT** - пополнение (Credit Card, PayPal и т.д.)
- **PURCHASE** - покупка произведения
- **ARTIST_EARNING** - заработок художника

### 4. Безопасность
- Проверка баланса перед покупкой
- Транзакционная целостность
- Валидация сумм
- Аутентификация для всех операций

## 🚀 Как запустить

### 1. База данных
```bash
psql -U postgres -d your_database
\i 'путь/к/add_wallet_system.sql'
```

### 2. Backend
```bash
cd backend
mvn spring-boot:run
```

### 3. Frontend
```bash
cd frontend
npm start
```

### 4. Тестирование
1. Откройте http://localhost:3000
2. Зарегистрируйтесь
3. Wallet → Add Funds → $100
4. Artworks → Purchase
5. Проверьте историю транзакций
