## 🔄 Логика работы

### Пополнение кошелька:
```
Wallet → Add Funds → Выбор способа → Сумма → Подтверждение
→ Баланс обновлен → Транзакция DEPOSIT создана
```

### Покупка произведения:
```
Artwork → Purchase → Проверка баланса
→ Если достаточно: Списание → Транзакция PURCHASE
→ Художник получает → Транзакция ARTIST_EARNING
→ Произведение продано
```

### Недостаточно средств:
```
Artwork → Purchase → Недостаточно баланса
→ Предупреждение → Кнопка "Add Funds"
→ Переход в Wallet → Пополнение → Возврат к покупке
```
